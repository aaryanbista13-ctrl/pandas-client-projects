\# Student Performance Analysis



\# Project Overview:



==>This project analyzes a dataset of student academic performance using Python and Pandas.



==>The program cleans the dataset, calculates total marks and averages, assigns grades, determines pass/fail     status, and generates useful insights about student performance.



\#Technologies Used

\- Python

\- Pandas



\#Dataset

==>The dataset contains:

\- Student Name

\- Math Marks

\- Science Marks

\- English Marks

\- Attendance



\#Features

\- Remove duplicate records

\- Calculate Total Marks

\- Calculate Average Marks

\- Assign Grades (A+, A, B, C, D, F)

\- Determine Pass/Fail Status

\- Find Highest Scorer

\- Find Lowest Scorer

\- Calculate Class Average

\- Calculate Subject-wise Average

\- Display Students Above Class Average

\- Display Students Below Class Average

\- Find Students with High Attendance

\- Find Students with Low Attendance

\- Show Grade Distribution

\- Calculate Pass Percentage

\- Export cleaned dataset as CSV
