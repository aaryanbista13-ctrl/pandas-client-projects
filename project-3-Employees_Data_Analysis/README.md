\# Employee Salary Analysis



\## Description



A Python + Pandas project that cleans employee salary data and generates useful HR insights.



\#Features



\- Remove duplicate records

\- Replace missing salary values

\- Convert data types

\- Find highest-paid employee

\- Find lowest-paid employee

\- Calculate average salary

\- Department-wise average salary

\- Department-wise total salary

\- Employees earning above average salary

\- Employees with rating above 4.5

\- Sort employees by salary

\- Display Top 3 highest-paid employees

\- Export cleaned CSV



\#Technologies



\- Python

\- Pandas
