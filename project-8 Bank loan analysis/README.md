\########Loan Application Analysis



\#Project Overview



==>This project analyzes loan application data using Python and Pandas. The objective is to help a banking client understand loan approvals, customer borrowing behavior, branch performance, and loan distribution.



\#Features



\- Remove duplicate records

\- Calculate total approved loan amount

\- Calculate average approved loan amount

\- Find highest and lowest approved loans

\- Display top 5 approved loans

\- Branch-wise loan amount analysis

\- Identify highest performing branch

\- Count loans handled by each branch

\- Revenue generated by each loan type

\- Find the most popular loan type

\- Top 5 customers by approved loan amount

\- Average credit score analysis

\- Highest credit score

\- Loan approval and rejection percentages

\- Customers with above-average loan amounts

\- Customers with below-average loan amounts



\#Technologies Used



\- Python

\- Pandas
