#Pandas Client Projects

A collection of Python projects built using **Pandas** to simulate real-world client requests involving data cleaning and business analysis.

## Skills Demonstrated

- Reading CSV files
- Data Cleaning
- Removing Duplicate Records
- Handling Missing Values
- Data Type Conversion
- Data Analysis
- GroupBy Operations
- Exporting Cleaned Data

****Projects***

#Project 1 – Client Data Cleaning
- Remove duplicate rows
- Handle missing values
- Convert data types
- Find the highest-priced product
- Calculate average prices

#Project 2 – Shop Inventory Analysis
- Analyze inventory data
- Find cheapest and most expensive products
- Calculate category-wise average prices
- Calculate total stock

#Technologies Used
