#Shop-Inventory-Analysis

====Objective====

Analyze a local shop inventory dataset after cleaning the data.

#Features

- Remove duplicate rows
- Replace missing values
- Convert data types
- Find the cheapest product
- Find the most expensive product
- Calculate category-wise average prices
- Calculate total stock available
- Export the cleaned dataset

#Concepts Used

- groupby()
- sum()
- mean()
- idxmax()
- idxmin()
- astype()
- replace()
- to_csv()
