# Client Data Cleaning

#Objective

The goal of this project is to clean a client's product dataset and generate useful business insights.

#Features

- Remove duplicate records
- Handle missing values
- Convert data types
- Calculate average product price
- Find the highest-priced product
- Export the cleaned dataset

#Concepts Used

- pandas.read_csv()
- drop_duplicates()
- replace()
- astype()
- mean()
- idxmax()
- to_csv()
